Providing a compelling experience for users of the MOC

Project logistics

  • Mentor: Jon Bell. Send an email to jbell-at-bu-dot-edu if you are interested in the project.
  • Min-max team size: 3-5
  • Expected project hours per week (per team member): 6-8
  • Will the project be open source? Yes, we will hope to release this under Apache license as part of OpenStack

Preferred past experience

Some experience in one or more of the following is preferred in possible team members:

  • Python
  • Django
  • HTLM
  • JavaScript
  • Service Deployment

What is the MOC-UI?

  • MOC-UI is the front end GUI for the Massachusettes Open Cloud
  • Provides a friendly, straight forward experience to new or returning cloud users
  • Includes a marketplace for purchasing resources and project management page for controlling vms
      While there is infinate number of projects, a few samples include:
    • spin off per-tenant UI for performance and security isolation
    • add other services, e.g., Trove, Heat
